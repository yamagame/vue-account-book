以下のようなCSVファイルをペーストすると集計できます。

```cs

2022/01/01,資産:売掛金,繰越,4000,,,,,,
,,,,,,,,,
2022/01/01,資産:消耗品,負債:買掛金,1000,材料,,,,,
2022/01/02,資産:新聞図書,資産:現金,2000,参考書,,,,,
,,,,,,,,,
2022/02/01,資産:消耗品,負債:買掛金,1000,材料,,,,,
2022/02/02,資産:新聞図書,資産:現金,2000,参考書,,,,,
,,,,,,,,,

```
